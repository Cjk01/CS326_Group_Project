User(id, username, followers, following, metadata)

Class representing a User (of Cachely!)

Constructor

new User(id, username, followers, following, metadata)

Creates a User
Parameters:
Name Type Default Description
id Number null a unique identifier
username string null the username text
followers Array.<Number> null stores followers of the User
following Array.<Number> null stores those being followed by the User
metadata Object Object mapping Deck IDs to study information Study information is an object including three keys timeLastStudied - The time in milliseconds since the Epoch at the time the deck was last studied timesStudied - The number of times the deck has been studied, to be used for spatial repetition beingStudied - A boolean indicating whether the deck is being actively studied
